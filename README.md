# DB손해보험 네이버 브랜드검색 URL 생성기

## 기능
- **1단계**: 소재명/보종/템플릿/랜딩 설정 → 에카 업로드용 CSV + 키워드ID 누적 CSV 다운로드
- **2단계**: 에카 결과 파일 업로드 → 최종 URL 엑셀 다운로드

## 로컬 실행
```bash
npm install
npm start
```

## Vercel 배포 방법
1. https://github.com 에서 새 레포 생성 (`db-url-tool`)
2. 이 폴더 전체를 올리기
3. https://vercel.com 가입 → GitHub 연결 → Import
4. 설정 그대로 Deploy 클릭
5. 생성된 URL 팀원들과 공유!
